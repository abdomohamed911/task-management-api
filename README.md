# Task Management API

A production-grade REST API for task management built with Node.js and Express, featuring comprehensive testing, documentation, and error handling.

## Features

- **RESTful Design**: Follows REST principles with proper HTTP methods and status codes
- **Validation**: Request body validation with detailed error messages
- **Pagination**: Built-in pagination with configurable page size
- **Filtering**: Filter tasks by status and priority
- **Error Handling**: Comprehensive error handling with consistent response format
- **Testing**: Unit and integration tests with high coverage
- **Documentation**: Complete API specification
